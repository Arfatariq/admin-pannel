# Restaurant POS Admin Panel

A Flutter-based Restaurant POS Admin Panel designed to simplify restaurant management. The application enables administrators to manage menu items, monitor customer orders, track inventory, and analyze restaurant performance through an interactive dashboard.

## Overview

The admin panel is integrated with the Restaurant POS mobile application, allowing administrators to perform real-time management of restaurant operations using Firebase services.

## Features

### Dashboard

* View total orders
* Monitor inventory status
* Display restaurant statistics
* Interactive sales and performance charts
* Quick overview of business activity

### Menu Management

* Add new food items
* Edit existing menu items
* Delete menu items
* Update product information, including:

  * Name
  * Description
  * Price
  * Category
  * Product Image

### Inventory Management

* View available inventory
* Update stock information
* Remove unavailable products
* Monitor inventory levels

### Order Management

* View customer orders
* Track incoming orders
* Monitor order history
* Manage order information

### Analytics

* Sales reports
* Order statistics
* Revenue visualization
* Performance graphs

### Authentication

* Secure administrator login
* Firebase Authentication integration
* Restricted access for authorized administrators


```bash

```
